In this chapter, we will explore two fundamental data structures in JavaScript: Arrays and Objects. These data structures are crucial for storing and managing data efficiently in your web applications.

> 4.1 Arrays

> 4.1.1 Introduction to Arrays

Arrays are ordered lists that can hold a collection of items. These items can be of any data type, including numbers, strings, objects, or even other arrays. Arrays in JavaScript are versatile and commonly used for various purposes, such as storing data, iterating over elements, and performing various operations.

**Declaring an Array**

You can declare an array in JavaScript using square brackets **[]**:

```js
const myArray = [1, 2, 3, 4, 5];
```

> 4.1.2 Common Array Methods

Arrays come with a variety of built-in methods to manipulate and access their elements efficiently. Here are some of the most commonly used array methods:

> 4.1.2.1 **push()**

The **push()** method adds one or more elements to the end of an array and returns the new length of the array.

```js
const fruits = ['apple', 'banana'];
fruits.push('cherry'); // ['apple', 'banana', 'cherry']
```

> 4.1.2.2 **pop()**

The **pop()** method removes the last element from an array and returns that element.

```js
const fruits = ['apple', 'banana', 'cherry'];
const lastFruit = fruits.pop(); // 'cherry', fruits is now ['apple', 'banana']
```

> 4.1.2.3 **forEach()**

The **forEach()** method executes a provided function once for each array element.

```js
const numbers = [1, 2, 3, 4, 5];
numbers.forEach(function (number) {
  console.log(number);
});
// Output:
// 1
// 2
// 3
// 4
// 5
```

> 4.1.2.4 **map()**

The **map()** method creates a new array with the results of calling a provided function on every element in the array.

```js
const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = numbers.map(function (number) {
  return number * 2;
});
// doubledNumbers is now [2, 4, 6, 8, 10]
```

> 4.2 Objects

> 4.2.1 Introduction to Objects

Objects are another essential data structure in JavaScript. Unlike arrays, which use numeric indices, objects use named properties to store and access values. Objects are used to represent structured data and are highly flexible.

**Declaring an Object**

You can declare an object in JavaScript using curly braces **{}**:

```js
const person = {
  firstName: 'John',
  lastName: 'Doe',
  age: 30
};
```

> 4.2.2 Accessing Object Properties and Methods

You can access object properties using dot notation (**object.property**) or bracket notation (**object['property']**).

```js
console.log(person.firstName); // 'John'
console.log(person['lastName']); // 'Doe'
```

Objects can also have methods, which are functions defined as properties of the object:

```js
const calculator = {
  add: function (a, b) {
    return a + b;
  },
  subtract: function (a, b) {
    return a - b;
  }
};

const sum = calculator.add(5, 3); // 8
const difference = calculator.subtract(8, 3); // 5
```
