Quiz App is a React-based Quiz website that allows users to select category, difficulty, type, and amount of questions.

---

## üçø Features

- Category: Users can select the category of question
- Difficulty: Users can select the difficulty of question example Easy, Medium, and Hard.
- Type: Users can select the type (Multiple Choices, or True/False) of Question.
- Amount of Questions: Users can determine the number of questions independently.

---

## üçø Technology

Quiz App is built using the following technologies:

- ReactJS
- Tailwind CSS
- Redux

---

## How to Start this project

### 1. Clone using git

```git
https://github.com/dj1samsoe/quiz-app.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run server

```bash
npm start
```

Open http://localhost:3000 with your browser to see the result.
