## Description

This Frontend Project is used for Simple Dashboard projects. That Consume API Login and Register from Backend.

## Tech Stacks

1. Vite
2. Vue JS 3
3. JavaScript

---

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
