## Introduction

Is a Simple Responsive Website for Assignment in RevoU Fundamental Course Software Engineering for 2 weeks (21 Aug 2023 - 1 Sep 2023).

## Features

**Data AOS**

This Website using Data AOS for Animation of Scroll

**Home Page**

This Home Page can be Dynamic Page in Input Name using JavaScript with Modal and Button. And there is a Contact Form that can show with Modal the result after submit button onclick also using JavaScript.

**Profil Page**

This Profil Page show content About Us and Vission-Mission

**Achievement Page**

This Page show about Alumni and Simple Traffic grid about the course. The content of story card I fill using JavaScript and I use pagination to show 3 cards of section
