### Deskripsi Proyek:

Global Planten adalah website marketplace untuk kebutuhan ekspor-impor tanaman dan produk agro lainnya. Proyek ini memerlukan sistem backend yang dapat menyimpan data produk, supplier, dan inquiries secara terstruktur serta mudah dicari dan difilter oleh pengguna.

### Fitur yang Dikerjakan:

1. Implementasi Custom Post Types untuk Supplier, Inquiries, dan Logistics.
2. Penggunaan Advanced Custom Fields (ACF) untuk input data khusus seperti kode referensi, klasifikasi, negara pembeli, dll.
3. Sistem filter data dinamis dengan Elementor + Taxonomy Filter.
4. Desain layout katalog dengan sidebar untuk pencarian & kategori.
5. Website bilingual (Bahasa Indonesia & English) untuk cakupan internasional.

### Keahlian yang Ditonjolkan:

- Penggunaan ACF dan CPT untuk kebutuhan data terstruktur.
- Integrasi Elementor dengan data dinamis dari WordPress backend.
- Optimasi UI untuk website bisnis B2B di sektor ekspor-impor.
